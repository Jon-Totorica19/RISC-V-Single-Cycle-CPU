## The full base RV32I (32-bit integer) instruction set to be implemented:

**R-type (10):** add, sub, sll, slt, sltu, xor, srl, sra, or, and 
**I-type — ALU-immediate (9):** addi, slti, sltiu, xori, ori, andi, slli, srli, srai
**I-type — loads (5):** lb, lh, lw, lbu, lhu
**I-type — jalr (1):** jalr
**S-type (3):** sb, sh, sw
**B-type (6):** beq, bne, blt, bge, bltu, bgeu
**U-type (2):** lui, auipc
**J-type (1):** jal

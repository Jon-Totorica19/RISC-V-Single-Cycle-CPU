## The RV32I (32-bit integer) instruction set to be implemented (Full implementation excluding LB, LH, LBU, LHU, SB, and SH):

**R-type (10):** add, sub, sll, slt, sltu, xor, srl, sra, or, and 
**I-type — ALU-immediate (9):** addi, slti, sltiu, xori, ori, andi, slli, srli, srai
**I-type — loads (5):** lw
**I-type — jalr (1):** jalr
**S-type (3):** sw
**B-type (6):** beq, bne, blt, bge, bltu, bgeu
**U-type (2):** lui, auipc
**J-type (1):** jal
